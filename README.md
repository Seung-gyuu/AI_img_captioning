# Image Captioning Application

## img_captioning_app : 
Users can upload an image and get captions using BLIP-2 model and Gradio interface.

## auto_url: 
Enter a url, program will extract up to 5 images from the web page and generate captions for them.

### Features

- Image captioning using the BLIP model
- User-friendly web interface with Gradio
- Simple and fast AI-powered caption generation
